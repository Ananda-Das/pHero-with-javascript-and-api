Question-1: Discuss the scope of 'var', 'let', and 'const'?

Solution: To know about the Socpe of Var, Let and Const we frist know what is Scope. Scope means where we can use these variables. Scope are two types. Globally and locally.

  1. Var: When var variable declared outside of a function then it act like global variable. If it declared inside a function then it act as local variable. But it also allow hosting. Which means if we call it inside a function sometime it can be call outside from the function.
  2. Let: Variables declared with let are block-scoped. The let variable used inside the block where it declared & it can used to the declared nested block. We can reasign the let variable.
  3. Const: Const variables are also block-scoped variable. As like let variable const variable can also use in the block & it's nasted block. We cannot reasign the const variable.

Question-2: Tell us the use cases of 'null' and 'undefined'?

  1. null: null is an assignment value for variable. If a variable assigned as null then it contains no value. When null is assigned any value then it means the variable is empty or blank.
  2. undefined : undefined means Not defined. If we decleare a variable but do not assigne any value then it shows us undefined.

Question-3: What do you mean by REST API?

Applications or devices can connect to and communicate with one another according to a set of rules known as an API, or application programming interface. An API that adheres to the design tenets of the REST, or representational state transfer, architectural style is known as a REST API. Because of this, REST APIs are also known as RESTful APIs.
